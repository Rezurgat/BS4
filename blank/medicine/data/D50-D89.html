Diseases of the blood and blood-forming organs and certain disorders involving the immune mechanism D50-D89

Type 2 Excludes
Type 2 Excludes Help
A type 2 excludes note represents "not included here". A type 2 excludes note indicates that the condition excluded is not part of the condition it is excluded from but a patient may have both conditions at the same time. When a type 2 excludes note appears under a code it is acceptable to use both the code (D50-D89) and the excluded code together.
